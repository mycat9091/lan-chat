# 🌐 局域网聊天室

基于 Node.js + Socket.io 的局域网实时通信工具，支持文字、图片、文件传输。

## ✨ 功能

- 💬 实时多人文字聊天
- 🖼️ 图片发送与预览
- 📁 文件传输与下载
- 👥 在线用户列表（支持折叠）
- 📜 历史消息同步（最近 100 条）
- 💾 自动记住上次昵称
- 📱 响应式设计，支持手机访问

## 📋 系统要求

- Node.js 14.0+
- 现代浏览器（Chrome、Firefox、Edge、Safari）

## 🚀 快速开始

### 1. 部署

```bash
# 克隆项目
git clone https://github.com/mycat9091/lan-chat.git
cd lan-chat

# 安装依赖
npm install
```

也可以双击 `manager.bat`，选择 **7. 部署（安装依赖）** 自动完成。

### 2. 启动服务器

**推荐：使用管理器**

双击 `manager.bat` 或运行：
```bash
node manager.js
```

管理器功能：启动 / 停止 / 重启服务器、查看状态、查看 IP、在浏览器中打开、一键部署。

**或直接启动：**
```bash
npm start
```

### 3. 访问

| 设备 | 地址 |
|------|------|
| 本机 | `http://localhost:3000` |
| 局域网其他设备 | `http://[服务器IP]:3000` |

> 在管理器中选择"查看本机 IP 地址"可快速获取局域网地址。

## 📖 使用说明

- **加入聊天**：输入昵称后点击"加入聊天"或按 Enter
- **发送消息**：底部输入框输入后按 Enter 或点击"发送"
- **发送图片**：点击 🖼️ 图标选择图片，点击聊天中的图片可全屏预览
- **发送文件**：点击 📎 图标选择文件，接收方点击即可下载
- **在线用户**：点击 👥 按钮可折叠/展开在线用户面板
- **清除记忆**：登录页点击"清除记忆"可重置保存的昵称
