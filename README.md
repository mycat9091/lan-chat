# 🌐 局域网聊天室

基于 Node.js + Socket.io 的局域网实时通信工具，支持文字、图片、文件传输，无需公网，开箱即用。

## ✨ 功能

- 💬 实时多人文字聊天
- 🖼️ 图片发送与全屏预览
- 📁 文件传输与一键下载（持久化存储在服务器磁盘）
- 👥 在线用户列表（支持折叠）
- 📜 历史消息同步（最近 100 条）
- 💾 自动记住上次昵称
- 📱 响应式设计，手机电脑均可访问

## 📋 系统要求

- Node.js 14.0+
- 现代浏览器（Chrome、Firefox、Edge、Safari）

## 🚀 快速开始

### 方式一：使用管理器（推荐）

双击 `manager.bat`，选择相应功能：

| 选项 | 功能 |
|------|------|
| 1 | 启动服务器 |
| 2 | 停止服务器 |
| 3 | 重启服务器 |
| 4 | 查看服务器状态 |
| 5 | 查看本机 IP 地址 |
| 6 | 在浏览器中打开 |
| 7 | 部署（安装依赖） |

**首次使用请先选 7 安装依赖，再选 1 启动服务器。**

### 方式二：命令行

```bash
# 安装依赖
npm install

# 启动服务器
npm start
```

### 访问地址

| 设备 | 地址 |
|------|------|
| 本机 | `http://localhost:3000` |
| 局域网其他设备 | `http://[服务器IP]:3000` |

> 在管理器中选择「5. 查看本机 IP 地址」可快速获取局域网地址。

## 📖 使用说明

- **加入聊天**：输入昵称后点击"加入聊天"或按 Enter
- **发送消息**：底部输入框输入后按 Enter 或点击"发送"
- **发送图片**：点击 🖼️ 图标，点击聊天中的图片可全屏预览
- **发送文件**：点击 📎 图标，接收方点击消息即可下载，文件存储在项目 `uploads/` 目录
- **在线用户**：点击 👥 按钮可折叠/展开在线用户面板
- **清除记忆**：登录页点击"清除记忆"可重置保存的昵称
